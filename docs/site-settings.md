# Site Settings

The **Site Settings** page controls site-wide content that appears across multiple pages (header ribbon, footer, contact info, social links).

## How to Access

1. Log in to WordPress admin
2. Click **Site Settings** in the left sidebar (near the top, with the gear icon)

## Fields

### Branding

| Field | Where It Appears | Default |
|-------|-----------------|---------|
| Site Slogan | Blue ribbon above the header on every page | ACTIVE. HEALTHY. OUTDOORS. |

### Contact Info

| Field | Where It Appears | Default |
|-------|-----------------|---------|
| Office Email | Footer "Connect" column | info@enduranceunited.org |
| Office Phone | Footer "Connect" column | (612) 850-3937 |
| Office Address | Footer "Connect" column | 713 Minnehaha Ave. East, Suite 216, Saint Paul, MN 55106, USA |

### Social URLs

| Field | Where It Appears |
|-------|-----------------|
| Facebook URL | Footer "Connect" column |
| Instagram URL | Footer "Connect" column |
| YouTube URL | Footer "Connect" column |

### Footer Content

| Field | Where It Appears |
|-------|-----------------|
| Footer About Text | Footer "About Us" column (first column) |

### Footer Links — Get Involved

Up to **4 links** displayed in the footer "Get Involved" column. Each link has:

- **Label** — the visible link text (leave blank to hide)
- **URL** — where the link goes

### Footer Links — Our Team

Up to **4 links** displayed in the footer "Our Team" column. Each link has:

- **Label** — the visible link text (leave blank to hide)
- **URL** — where the link goes

## Tips

- All fields have sensible defaults. You only need to fill in what you want to change.
- Leave a footer link's **Label** blank to hide that slot entirely.
- Changes take effect immediately after clicking **Update**.
