# Managing Testimonials

Testimonials appear in two places:
1. **Homepage slider** — featured testimonials rotate automatically
2. **Testimonials page** (`/testimonials/`) — all testimonials displayed in a card grid

## How to Access

1. Log in to WordPress admin
2. Click **Testimonials** in the left sidebar (quote icon)

## Adding a New Testimonial

1. Click **Testimonials > Add New Testimonial**
2. Fill in the following:

| Field | Purpose | Example |
|-------|---------|---------|
| **Title** | Internal reference only (not displayed on the site) | "Sarah M. - Nordic 2025" |
| **Editor content** | The testimonial quote text | "Endurance United completely changed my approach to training..." |
| **Author Name** | Person's name shown below the quote | Sarah M. |
| **Program / Role** | Context shown below the author name | Adult Nordic Program |
| **Show on Homepage** | Check to include in the homepage testimonial slider | (checkbox) |

3. Click **Publish**

## How It Appears

### Homepage Slider
- Only testimonials with **Show on Homepage** checked appear here
- Up to 10 featured testimonials are shown
- They auto-rotate every 6 seconds
- Navigation dots allow manual selection

### Testimonials Page
- All published testimonials appear here regardless of the featured checkbox
- Displayed in a responsive card grid
- Each card shows the quote, author name, and program/role

## Tips

- Write the quote in the main editor. Keep it concise (2-4 sentences works best for the homepage slider).
- The homepage slider works best with 3-6 featured testimonials.
- The title field is just for your reference in the admin list — use something that helps you identify the testimonial quickly.
